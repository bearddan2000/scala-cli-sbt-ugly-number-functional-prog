# scala-cli-sbt-ugly-number-functional-prog

## Description
Divide the number by greatest divisible powers of
2, 3 and 5, if the number becomes 1 then it is an
ugly number otherwise not.
Follows functional programming practices.

## Tech stack
- scala
- millbuild

## Docker stack
- nightscape/scala-mill

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
